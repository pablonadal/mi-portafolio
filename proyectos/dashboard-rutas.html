Dashboard de Rutas de Trabajo

Proyecto de visualización de datos con Power BI

Volver al inicio

Descripción

Este proyecto consiste en el desarrollo de un dashboard interactivo con Power BI, diseñado para visualizar y analizar las rutas de trabajo efectivamente realizadas por los equipos técnicos. A través de la comparación entre estas rutas reales y las rutas óptimas definidas por Google Maps, la herramienta permite identificar ineficiencias logísticas y estimar el desperdicio de combustible generado por desvíos innecesarios o decisiones subjetivas en el recorrido.

Tecnologías utilizadas

  • Power BI
  • Python (ETL)
  • Google Maps API
  • MyGeotab
  • Pandas
  • SQL
