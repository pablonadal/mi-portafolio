Sistema de Ruteo Inteligente

Proyecto de ordenamiento de rutas y notificaciones automáticas

Volver al inicio

Descripción

Sistema inteligente que optimiza rutas de trabajo según la cantidad de móviles y distancias a recorrer, predice tiempos de llegada considerando la actividad a realizar, y envía notificaciones automáticas por WhatsApp a los choferes y por email a los clientes con el horario estimado de visita.

Tecnologías utilizadas

  • Gmail API
  • Python
  • Google Maps API
  • OR-Tools
  • FastAPI
  • WhatsApp API
  • Pandas
  • PostgreSQL
